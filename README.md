# The-Slime-Knight

It is 2D Platformer mini game made on godot 4.





Godot 4 Project Import \& Export Guide (Windows)



1\. Install Godot 4

\- Download the latest version of Godot 4 from the official website

\- Extract the files if needed and launch the Godot executable



2\. Import the Project

\- Open Godot

\- Click on "Import" in the Project Manager

\- Navigate to your project folder

\- Select the project.godot file

\- Click "Import \& Edit"



3\. Open and Verify the Project

\- Ensure the project loads correctly

\- Run the project using the Play (▶) button to confirm everything works



4\. Configure Export Settings

\- Go to Project -> Export

\- Click "Add..." and select Windows Desktop

\- Download export templates if prompted

\- Set:

&#x20; - Export path (where the build will be saved)

&#x20; - Executable name

&#x20; - Optional settings like icon or fullscreen



5\. Export the Project

\- Click "Export Project"

\- Choose a destination folder

\- Godot will generate a .exe file along with required files



6\. Run the Game

\- Open the exported folder

\- Double-click the .exe file to play the game
